Create a full-stack web application called **KiwiQA Online Examination Portal**.

This application is for conducting Aptitude and General Exams for candidates applying to KiwiQA Services Pvt Ltd.

TECH STACK:

* Backend: Node.js + Express
* Database: PostgreSQL
* ORM: Drizzle ORM
* Authentication: Email OTP (print OTP in console for development)
* Password hashing using bcrypt
* JWT authentication
* Frontend: Simple responsive Bootstrap UI

ROLES:

1. Admin
2. Student

====================================
AUTHENTICATION SYSTEM
=====================

* Student registers with:

  * Full Name
  * Email
  * Password

* On registration:

  * Generate 6-digit OTP
  * OTP expires in 5 minutes
  * Store OTP in database
  * Print OTP in console (development mode)
  * User must verify OTP before login

* Limit OTP requests to 3 per hour

* Login only allowed if verified
